# TernakOS Landing Page

High-converting, modern landing page for TernakOS, built with React, Tailwind CSS, and Framer Motion.

## 🚀 Fitur Utama
- **100% Bahasa Indonesia**: Antarmuka yang ramah bagi peternak lokal.
- **Modern & Animatif**: Transisi halus menggunakan Framer Motion.
- **Responsif**: Dioptimalkan untuk HP Android dan Desktop.
- **Design System Premium**: Menggunakan palet hijau profesional dan font Plus Jakarta Sans.

## 🛠️ Teknologi
- **Framework**: React 19 (Vite)
- **Styling**: Tailwind CSS 4
- **Animasi**: Framer Motion
- **Ikon**: Lucide React
- **Routing**: React Router 7

## 📦 Cara Menjalankan Secara Lokal

1. **Install dependensi**:
   ```bash
   npm install
   ```

2. **Jalankan server pengembangan**:
   ```bash
   npm run dev
   ```

3. **Buka di browser**:
   Kunjungi [http://localhost:5173](http://localhost:5173)

## 📄 Struktur Proyek
- `src/components/`: Komponen UI (Navbar, Footer).
- `src/sections/`: Bagian utama halaman (Hero, Fitur, Harga, dll).
- `src/pages/`: Halaman landing dan placeholder Auth.
- `src/index.css`: Konfigurasi tema Tailwind v4.
